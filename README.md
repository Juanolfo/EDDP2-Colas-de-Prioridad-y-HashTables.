# Simulador de Cola de Impresión
Simulador interactivo en Java para Sistemas Operativos que gestiona prioridades de documentos utilizando Montículos Binarios, Tablas Hash y Listas Enlazadas.
Cuenta con una interfaz gráfica completa y visualización en tiempo real del árbol jerárquico para el análisis de las estructuras de datos en ejecución.
